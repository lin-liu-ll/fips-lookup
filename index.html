Find the FIPS code for any U.S. county, or look up a county and state from a known FIPS code.

Federal Information Processing Standard (FIPS) codes are numbers that uniquely identify U.S. geographic areas. The commonly used 5-digit county FIPS code combines a 2-digit state code and a 3-digit county code. For example, Iowa’s state code is 19, Des Moines County’s code is 057, making the full county FIPS code 19057.
